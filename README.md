# urbandict-rb
A Ruby interface to Urban Dictionary API

## Installation
### RubyGems
Execute:

```shell
$ gem install urbandict
```

### Bundler
Add this line to the application's Gemfile:

```ruby
gem('urbandict')
```

Then execute:

```shell
$ bundle
```
